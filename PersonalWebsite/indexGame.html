Projects

Jeremy Cohen

Jeremy Cohen is a dungeon crawler made by Unity3D . The player has the option to stop time and que actions in order to execute them in a satisfying flurry of devastating blows. The player customizes his play style as he goes through procedurally generated levels. C# and Maya are some of the technologies we used to create this game.

MERCy Frontier


MERCy frontier is a game that aims to incorporate well-known turn-based strategy paradigm into the boardgame medium. The players gather resources, build different buildings and recruit mercenaries and buy spells from a common pool. You can earn extra powers by holding the power zones spread around the battleground. If you want to learn more about the game you can find the details here .

Fallout Wars


Fallout Wars is a social game that focuses on bluffing, intimidation and deception. In each round the players bet their survivors (health and score) to earn additional resources that give them guns to fight other players, food to create more survivors, monuments as long term investments and tech as buff cards. The catch is you know the resource type but you don't know the resource value. (that gun be 1 power or 3 power etc.) In the discussion round, players take turns making demands from each other and in the end destroy all other players. If you want to learn more about the game you can find the details here .

The Four Knights

Four Knights is a card game that I designed myself. It has a special place in my heart because it was the first game I ever designed. A few years, ago I decided to port the game to PC using Python and TKinter. Read more about the rules here.
